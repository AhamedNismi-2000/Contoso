# Contoso Sales Data Analysis Project

## Overview
This project is a complete Data Analytics workflow built using the Contoso Dataset.  
The project focuses on data cleaning, transformation, analysis, and dashboard visualization using industry-standard tools.

## Tools & Technologies
- SQL
- Python
- Excel
- Power BI

---

# Project Workflow

## 1. Data Source
The project uses the Contoso retail dataset which includes:
- Sales data
- Customer data
- Product data
- Store data
- Geography data

---

## 2. SQL Data Processing
SQL was used for:
- Data cleaning
- Joins between tables
- Aggregations
- KPI calculations
- Data transformation

Example analysis:
- Total sales
- Monthly revenue
- Product performance
- Regional sales analysis

---

## 3. Python CSV Generation
Python was used to:
- Connect with SQL database
- Execute SQL queries
- Export query results into CSV files

### Python Libraries Used
```python
pandas
sqlite3 / pyodbc
csv
```

---

## 4. Excel Usage
Excel was used for:
- Data inspection
- Data validation
- Checking missing values
- Basic preprocessing

Power BI was used to create:

- Interactive visualizations

# How to Run the Project

## SQL
Run the SQL scripts using your preferred SQL environment.

## Python
Install required libraries:

```bash
pip install pandas
```

Run the Python script:

```bash
python export_csv.py
```

## Power BI
Open the `.pbix` file using Power BI Desktop.
